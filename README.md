# README

This README documents steps necessary to get the
application up and running.

## System requirements:

* Ruby -v 3.2.2

* SQlite

* Node -v 16.x.x

## Setup

Run `bin/setup` to install dependencies and prepare a seeded databse

## Development

Run `bin/dev` to start a development server
